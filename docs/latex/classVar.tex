\doxysubsection{Detailed Description}
encapsulates information about vars 

The \doxylink{classVar}{Var} class contains information about vars, and a way to interface with vars\+\_\+raw and vars\+\_\+tree. A cannonical (yes it is misspelled, deal with it) \doxylink{classVar}{Var} is one that defines how all vars with that name should work. Unless you are trying to change the vars\+\_\+raw file or need to know every last bit of information about a var, it\textquotesingle{}s not a good idea to cannonize it. Cannonizing a var involves calling the functions \doxylink{group__file_gae6175771ecd2ad72240d87c926069762}{get()} or \doxylink{group__file_ga61756da973a8fae33e2b71891d094dd9}{update()}.
