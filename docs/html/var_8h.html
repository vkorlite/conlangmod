Detailed Description

Header file for the Var class.

It is the header file for the Var class, which stores information about properties of an object, keeps track of what each Var is dependent on, and writes it to files. Full information on each Var is stored in vars_raw, which is read and written for cannonical vars. A tree describing all the dependencies is in vars_tree, managed by the tree functions.
